# Установка!:

* apt update
* apt upgrade
* pkg install python
* pkg install python3
* pip install requests
* pip install git
* git clone https://github.com/Artem450-ss/NanoBomber 
* cd NanoBomber 
* python3 nano.py 
* Готово!
